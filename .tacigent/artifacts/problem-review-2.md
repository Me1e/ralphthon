# Problem Review Round 2

## 발견한 문제
- failure moment가 약간 장황했다.
- `문장`이나 `재사용` 같은 표현이 solution artifact를 연상시켰다.

## 수정 사항
- `필요한 근거와 문장을 여러 문서와 사람 사이에서 수동으로 모은다` → `필요한 근거를 여러 문서와 사람 사이에서 수동으로 모은다` — solution scent를 줄였다.
- `한 번 만든 답이 다음 review에 재사용되지 못한다` → `만든 답이 다음 review로 안정적으로 이어지지 않는다` — 기능 설명보다 문제 현상에 가깝게 바꿨다.

## 이번 round에서 삭제한 것
- `문장` 표현 삭제: answer-generation tool처럼 읽히는 뉘앙스를 줄였다.
