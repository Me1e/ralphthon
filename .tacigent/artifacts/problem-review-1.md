# Problem Review Round 1

## 발견한 문제
- target user 표현이 아직 약간 넓었다: `small / Series A-B` 표기가 거칠고, 어떤 sales motion인지 바로 안 보였다.
- `조직`이라는 표현이 약간 크다: 실제 pain은 dedicated GRC headcount 부재에서 나온다.

## 수정 사항
- `small / Series A-B` → `small and Series A-B` — segment 표기를 더 읽기 쉽게 통일했다.
- `전담 GRC 조직` → `전담 GRC 인력` — 실제 부족 자원을 더 정확히 표현했다.
- `mid-market 또는 enterprise customer review` → `첫 10-50개의 mid-market 또는 enterprise review` — urgency와 buyer motion을 더 구체화했다.
